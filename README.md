# Keyboard Switcher

一键切换 Android 输入法（键盘）的快捷开关小部件。

## 使用

安装后会在快捷设置面板中添加一个磁贴（Quick Settings Tile）：

- 点击磁贴：在上次使用的输入法与当前输入法之间切换
- 长按磁贴：弹出输入法列表，可选择任意已安装的输入法

通过 [Shizuku](https://github.com/RikkaApps/Shizuku) 获取 `WRITE_SECURE_SETTINGS` 权限，直接写入系统设置来切换输入法，无需手动弹窗选择。

## 要求

- Android 14 (API 34) 及以上
- 已安装并运行 [Shizuku](https://github.com/RikkaApps/Shizuku)
